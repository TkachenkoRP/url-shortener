# Сервис коротких ссылок

Сервис для сокращения ссылок с поддержкой многопользовательского режима, лимитов переходов и времени жизни ссылок.

## Функции

- Создание уникальных коротких ссылок для каждого пользователя
- Ограничение количества переходов по ссылке
- Автоматическое удаление ссылок по истечении времени жизни
- Уведомления о событиях (истечение времени, превышение лимита)
- Идентификация пользователей по UUID
- CLI интерфейс для управления

## Использование

### Основные команды:

- `shorten <URL>` - Создать короткую ссылку
- `list` - Показать все ваши ссылки
- `open <code>` - Открыть ссылку в браузере
- `update <code> <limit>` - Изменить лимит переходов
- `delete <code>` - Удалить ссылку
- `user` - Информация о пользователе
- `help` - Справка
- `exit` - Выход

## Конфигурация

Настройки в `src/main/resources/application.properties`:

- `url.ttl.hours=24` - Время жизни ссылки в часах
- `url.max.clicks.default=100` - Лимит переходов по умолчанию
- `url.short.code.length=6` - Длина короткого кода
- `base.url=my` - Базовый URL для коротких ссылок

## Архитектура

- **Модель**: User, ShortUrl
- **Сервисы**: UrlService, UserService, NotificationService
- **Контроллер**: CliController для взаимодействия с пользователем
- **Утилиты**: UrlShortener для генерации коротких кодов
